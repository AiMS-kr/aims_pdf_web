# AIMS Leaflet

AIMS(Artificial Intelligence for Manufacturing System) 회사 리플렛 웹 뷰어입니다.

**배포:** https://aimsqr.netlify.app/

## 주요 기능

- 한국어/영어 언어 전환
- 이미지 캐러셀 뷰어 (클릭 시 전체화면 확대)
- PDF 다운로드

## 기술 스택

React 19 · react-responsive-carousel · @react-pdf/renderer · Netlify

## 실행

```bash
npm install
npm start
```
